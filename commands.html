Twitch Modifiers

Available Commands

!tm help

Sends a link in your chat to this page

!rotate

Changes the spawn rotation of the notes. (As if you were in 360 mode!)

  • left - Rotate notes to the left
  • right - Rotate notes to the right
  • home - Reset notes rotation to default
  • stop - Stop notes rotation

!scale

Changes the scale of the Notes & Bombs with a gradual increment of the size.

  • 0.5 - Set scale to 50%
  • 1.0 - Set scale to 100%
  • 1.5 - Set scale to 150%

!notedistract

Changes the notes and note arrows to be a random colour for a couple seconds.
