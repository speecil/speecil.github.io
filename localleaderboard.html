Local Leaderboard

Description

The LocalLeaderboard mod saves your results to a JSON file and displays them in-game. It shows the date, accuracy, score, and modifiers of each play. You can sort by date or high-score in ascending or descending order. If you have the "BeatLeader" mod installed, you can also watch the replay.

Features

  • Saves results from playing a level to a JSON file.
  • Displays results in a custom UI.
  • Shows date, accuracy, score, and modifiers.
  • Allows sorting by date or high score.
  • Ascending or descending order.

Usage

To use the Local Leaderboard mod, simply install it by unzipping the latest release and finding the LocalLeaderboard.dll file. Drag that file into your Beat Saber installs' Plugin folder. After installing, your results from playing a level will be saved automatically to a JSON file, and you will be able to access the custom UI within the game to view your results.
